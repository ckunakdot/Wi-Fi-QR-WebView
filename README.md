# Wi-Fi-QR-WebView
Control4 Wi-Fi QR WebView

The Wi-Fi QR WebView driver displays a QR code image on Control4 touchscreens, mobile apps, and other navigators. It features a built-in web server that serves a beautifully styled HTML page matching the Control4 aesthetic.

This driver is ideal for displaying Wi-Fi network credentials, guest access information, or any other QR code content. Simply replace the bundled QR code image with your own.

Features

•	Built-in Web Server: Runs on port 1111 to serve the QR code display page
•	Burn-in Prevention: Content automatically moves every 60 seconds to prevent screen burn-in
•	Customizable Styling: Configure colors for background, title text, and QR code container
•	Responsive Design: Automatically adapts to any screen size and orientation
•	Control4 Theme: Styled to match the Control4 interface aesthetic
•	Easy QR Code Replacement: Simply swap the PNG file in the driver package
•	Server URL Display: Shows the URL in Composer Pro for easy reference
